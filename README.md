# 🏥 Hospital Load Balancer System (Nalam)

A full-stack healthcare solution developed during a hackathon to efficiently distribute patient load across hospitals using real-time data and smart allocation.

## 📌 Overview

The system helps manage hospital overcrowding by providing a platform where patient requests can be balanced across multiple hospitals based on availability of resources such as beds, doctors, and facilities.

## 🚀 Features

* 📱 Android app for users to request hospital services
* 🌐 Web dashboard for monitoring hospital data
* 🔄 Real-time data updates using Supabase
* ⚙️ Python backend for handling logic and APIs
* 🏥 Intelligent load balancing across hospitals
* 🔐 User authentication and data management

## 🛠️ Tech Stack

### Mobile App

* Android (Java/Kotlin)

### Web Application

* React.js
* HTML, CSS, JavaScript

### Backend

* Python (Flask/FastAPI)

### Database

* Supabase

## 🧩 System Architecture

Android App ↔ Python Backend ↔ Supabase Database ↔ React Web Dashboard
